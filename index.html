HTML é a linguagem de marcação padrão para criar páginas da web

O que é HTML

  • HTML significa Hyper Texte Markup Language
  • HTML é a linguagem de marcação padrão para a criação de páginas da web
  • HTML descreve estrutura de uma página da web
  • HTML cosiste em uma série de elemento
  • Os elementos HTML informa ao navegador como exibir o conteúdo
  • Os elementos HTML rotulam partes do conteúdo coo "isto é um titulo","isto é u paragrafo","isto é um link",etc
